# Python OOP Lesson

Проект для изучения объектно-ориентированного программирования на Python.

## Модели данных

### **BaseProduct** 
- Базовый абстрактный класс с общей функциональностью для всех продуктов

### **PrintMixin**
- Реализован класс для логирования создания объектов
- При создании объекта автоматически выводится информация о классе и параметрах
- Пример вывода: `Product('iPhone 15', '512GB, Space Gray', 89999.99, 5)`

### **Product (Товар)**
Базовый класс для всех товаров в магазине.

**Атрибуты:**
- `name` — название товара
- `description` — описание товара
- `price` — цена товара
- `quantity` — количество товара на складе

---

### **Smartphone (Смартфон)**
Класс-наследник от `Product` для представления смартфонов.

**Наследует все атрибуты от `Product`**

**Дополнительные атрибуты:**
- `efficiency` — производительность устройства
- `model` — модель смартфона
- `memory` — объем встроенной памяти (в ГБ)
- `color` — цвет устройства

---

### **LawnGrass (Трава газонная)**
Класс-наследник от `Product` для представления газонной травы.

**Наследует все атрибуты от `Product`**

**Дополнительные атрибуты:**
- `country` — страна-производитель
- `germination_period` — срок прорастания
- `color` — цвет травы

---

### **Category (Категория)**
Класс для организации товаров по категориям.

**Атрибуты:**
- `name` — название категории
- `description` — описание категории
- `products` — список товаров в категории
