# python_oop_lesson
Проект для изучения объектно-ориентированного программирования на Python.
## Реализованная функциональность

### Модели данных
- **Product (Товар)**
  - Атрибуты: название, описание, цена, количество
  
- **Category (Категория)**
  - Атрибуты: название, описание, список товаров
## Установка
# Клонировать репозиторий
git clone https://github.com/semyon-shtelman/python_oop_lesson.git
# Установить зависимости
poetry install
